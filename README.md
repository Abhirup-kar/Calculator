# Scientific Calculator

A modern scientific calculator built with Python and PyQt5, featuring a sleek glassmorphism UI design.

## Features

- **Basic arithmetic**: addition, subtraction, multiplication, division
- **Scientific functions**: `sin`, `cos`, `tan`, `log`, `ln`, `sqrt`
- **Constants**: π (pi), e
- **Parentheses** support for complex expressions
- **Keyboard shortcuts** for quick input
- Translucent glass-style UI with drop shadow effects

## Requirements

- Python 3.x
- PyQt5

Install dependencies:

```bash
pip install PyQt5
```

## Usage

Run the calculator with:

```bash
python main.py
```

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9`, `+`, `-`, `*`, `/`, `.`, `(`, `)` | Input characters |
| `Enter` / `Return` | Evaluate expression |
| `Backspace` | Delete last character |
| `Esc` | Clear the display |
